## Installation

```bash

pip install numpy
```

## Usage

### Transmit Quantum Audio
```bash
python qwhisper.py send <host> <port>
```

### Receive Quantum Audio
```bash
python qwhisper.py receive <port>
```

### Example
```bash
# Terminal 1 - Receiver
python qwhisper.py receive 35235

# Terminal 2 - Transmitter
python qwhisper.py send localhost 35235
```
